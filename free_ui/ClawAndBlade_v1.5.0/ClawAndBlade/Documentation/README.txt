TECHNICAL SPECS / 技術規格
--------------------------

Characters / 角色:
  • 128x128 px & 256x256 px
  • Naming: character_001.png ~ character_040.png

Icons / 圖標:
  • 64x64 px each
  • Naming: category_###_name.png
  • Tilesheets: 8 columns × 6 rows, no spacing

Character Parts / 角色部件:
  • Body (25), Face (32), Hair Back (12), Hair Front (22)
  • Dual sizes: 128x128 + 256x256

Buildings / 建築物:
  • Various sizes, 4 color variants each
  • Individual PNGs + combined sheet

GUI / 介面:
  • Buttons: 64x64, 192x64, 192x192
  • Icons: 3 states (regular, pressed, disabled)
